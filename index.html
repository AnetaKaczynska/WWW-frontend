About us

Our page contains free statistic data divided into categories. We use CVS files and convert them into HTML tables, so you can quickly display them in a clear way, sort and filter them effectively.

How to use it?

Choose desired data from the menu above. As you scroll the page, more content will be load. If you wish to sort the table or apply the filters, you have to load the whole data, by clicking on 'Load All' button. Then you can click on column names to sort the table by them alternately in ascending and descending order. To filter the table use 'Search' window.
